Tagberry Qt5 client
-------------------

*Work in progress!*

## Dependencies

* C++17 compiler
* CMake
* Qt5

## Building

```
git clone https://github.com/tagberry/tagberry-qt.git
cd tagberry-qt
mkdir build
cd build
cmake ..
make -j
```

## Running

```
./bin/Debug/tagberry-qt
```
